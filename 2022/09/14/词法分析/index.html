词法分析

开发语言的选择

这次我选择的开发语言是 Java，主要是考虑到上学期在 OOP（21系面向对象程序设计）课程中使用的是 Java 语言，而且 Java 有很多很好用的包（库函数），可以给开发提供极大的便捷。

词法分析文件结构

逐文件分析

IOUtils

这是用来读取文件和输出到文件的工具类。

从文件读入

在词法分析中，需要注意的就是文件应该怎样读取，我这里的做法是使用 Scanner 按行读，然后每读一行添加\n，全部拼接起来。这样也避免了直接读整个文件导致不同系统换行符不一致（\r、\n、\r\n等多种情况）的问题。

使用 StringJoiner 可以方便的在读每一行的时候添加\n。

输出到文件

这没啥好说的。

TokenType

这是一个枚举。将所有的类别码全部放在这个枚举里面，方便后续词法分析的使用。

Token

这是关键词对象。提取的每个单词的相关信息都存放在这里。

Lexer

这是词法分析部分。

Compiler

这是程序的主入口，考虑到这是第一次作业需要输出词法分析的结果，我们使用 stage 来方便的控制输出内容。
